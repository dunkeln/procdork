The available chat summary mart is `procdork_analytics.main.mart_chat_summary`, a base table containing:

- Count metrics: `session_count`, `message_count`, `event_count`, `cited_source_count` (`BIGINT`, nullable)
- Time bounds: `first_session_at`, `last_session_at` (`TIMESTAMP WITH TIME ZONE`, nullable)
